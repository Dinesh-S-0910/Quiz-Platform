🗄️ DATABASE (MongoDB)
─────────────────────────────────────────────────────────────────────────────────

Database Name: quiz-platform

Collections:

1. quizzes
   - Stores all created quizzes
   - Fields: title, questions[], createdAt, updatedAt
   - Each question: text, 4 options, correct answer index

2. attempts
   - Stores all quiz attempts
   - Fields: username, quizId, score, totalQuestions, date
   - Records user performance history

❓ QUICK TROUBLESHOOTING
─────────────────────────────────────────────────────────────────────────────────

Problem: "Cannot find module 'express'"
Solution: Run "npm install"

Problem: "Port 3000 already in use"
Solution: Change PORT in app.js or close other application

Problem: "MongoDB connection error"
Solution: Start MongoDB with "mongod" command in separate terminal

Problem: "Page not loading"
Solution: Check server console for errors, restart server

👉 See SETUP_INSTRUCTIONS.md for detailed troubleshooting
