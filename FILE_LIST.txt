🎯 MAIN FILES EXPLAINED
═══════════════════════════════════════════════════════════════════════════════

1️⃣ app.js (Main Server)
   - Initializes Express application
   - Connects to MongoDB
   - Sets up middleware (static files, JSON parsing)
   - Mounts all routes
   - Starts server on port 3000

2️⃣ package.json (Dependencies)
   - Dependencies: express, mongoose, ejs
   - Dev Dependencies: nodemon
   - Scripts: npm start, npm run dev

3️⃣ config/database.js (MongoDB Connection)
   - Connects to local MongoDB
   - Default: mongodb://localhost:27017/quiz-platform
   - Error handling & logging

4️⃣ models/Quiz.js (Database Schema)
   - Defines quiz structure
   - Fields: title, questions, options, correctAnswer
   - Timestamps: createdAt, updatedAt

5️⃣ models/Attempt.js (Score Storage)
   - Stores quiz attempts
   - Fields: username, quizId, score, totalQuestions, date
   - Reference to Quiz collection

6️⃣ routes/quizRoutes.js (All Routes)
   GET  /                      Home page
   GET  /create                Quiz creation form
   POST /create                Save quiz
   GET  /quizzes               List quizzes
   GET  /quiz/:id              Display quiz
   POST /submit/:id            Calculate score
   GET  /attempts              Search form
   POST /attempts              Show history

7️⃣ views/ (EJS Templates)
   index.ejs              Home page with 3 buttons
   create.ejs             Dynamic quiz creation form
   quizzes.ejs            List of all quizzes
   quiz.ejs               Quiz questions for attempt
   result.ejs             Score display
   attempts.ejs           Username search form
   attemptHistory.ejs     Attempt history table

8️⃣ public/css/style.css (Styling)
   - Clean, minimal design
   - Responsive layout
   - 320+ lines of CSS
   - Multiple color schemes
   - Form, button, table styles
